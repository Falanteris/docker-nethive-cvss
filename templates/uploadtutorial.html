To upload a json parameter, you will have to utilize the POST and PUT request, and upload it from there. You can use tools such as 'curl' to allow you to send the json files for the calculation. e.g curl -X PUT -d params.json [summarizer's address]

List Of Parameters Description
vul The type of vulnerability in question, for now you can use SQLi or XSS as a parameter
host IP or domain of the victim host
url the attacked endpoint of the host.
